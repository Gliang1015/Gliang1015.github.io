refer反爬&MongoDB数据库

有时我们爬取数据时会出现refer错误，这就是我们将要学的新的反爬点：refer跳转。
并且在爬取数据后，我们通常会将数据存到数据库，方式使用和调用。

refer跳转

部分特殊url的请求，会检查你是从哪一个url跳转过来的。
比如我只允许我自己的网站访问我自己的图片服务器，那我的域名是www.google.com ，那么图片服务器每次取到Referer来判断一下是不是我自己的域名www.google.com ，如果是就继续访问，不是就拦截。

数据库

数据库用来保存数据的，可以理解成特殊的为文件。但相对于文本文件，数据库提取数据较为方便，可以利用命令语句快速找到符合条件的数据，也方便多台机器操作，最为重要的是，它的读取速度很快。

数据库的分类

  • 关系型数据库
    关系型数据库（Relational Database，简称 RDB）是一种以关系（即表格）为基础的数据库，它采用了关系代数等数学概念和方法来处理数据。其核心思想是将数据存储在由多个表格组成的关系中，每个表格包含多行数据和多个列（即属性），每个属性只有一种数据类型。
    • oracle mysql
  • 非关系型数据库
    非关系型数据库又被称为 NoSQL（Not Only SQL )，意为不仅仅是 SQL。通常指数据以对象的形式存储在数据库中，而对象之间的关系通过每个对象自身的属性来决定，常用于存储非结构化的数据。
    • mongodb redis

MongoDB数据库的操作

认识MongoDB

mongodb文档类似于json对象（是一种轻量级的数据交换格式），旨在为web应用提供可扩展的高性能数据存储解决方案。

库操作

  • 显示所有库：show dbs
    默认存在的三个数据库：admin、config、local。
  • 切换/创建数据库：use 数据库名称
  • 查看所在库：db
  • 删除库：db.dropDatabase()（先切换到数据库）

集合操作

  • 显示当前数据库的集合：show collections
  • 创建集合：db.createCollection(’name‘)
  • 删除集合：db.集合名称.drop()

文档操作

  • 添加一条文档：db.集合名称.insert({键值对})
  • 添加多条文档：db.集合名称.insert([{键值对},{},…])
  • 查询文档：db.集合名称.find()
  • 格式化显示：db.集合名称.find().pretty()
  • 修改文档：db.集合名称.update({原内容},{新内容}) 会将整条文档进行更改
  • 指定修改属性：set后面是修改的值 db.集合名称.update({},{$set:{}})
  • 更新所有满足条件的文档：{multi:true}
  • 删除文档：db.集合名称.remove({name:’叶子’}) 和修改一样，会将整条文档删除，并且会将所有符合条件的文档全部删除
  • 只删除符合条件的第一条文档：{justOne:true}
  • 将所有文档删除：db.集合名称.remove({})

id参数，mongodb文档都会有唯一的_id，当你不去设置时，mongodb会自动分配一个ObjectId。

条件

  • and条件：{$and:[{expression1},{expression2}]}
  • or条件：{$or:[{expression1},{expression2}]}

操作符

  • $ne：不等于
  • $gt：大于
  • $lt：小于
  • $gte：大于等于
  • $lte：小于等于

selenium介绍

selenium

selenium是一个web的自动化测试工具，最初是为网站自动化测试而开发的。
selenium可以直接运行在浏览器上，它支持所有主流的浏览器（包括phantomjs这些无界面的浏览器）可以接受指令，让浏览器自动加载页面，获取需要的数据，甚至页面截屏。
可以模拟事件运行js，从而获取信息。

PhantomJS

phantomjs是一个基于webkit的无界面浏览器，它会吧万盏加载到的内存并执行页面上的js。

Chromedriver

chromedriver也是一个能够被selenium驱动的浏览器，但是和phantomjs的区别在于它是有界面的。
我们想通过python代码模拟浏览器，打开浏览器，获取数据，就需要这样一个驱动去启动浏览器。
https://googlechromelabs.github.io/chrome-for-testing/
因为我使用的是虚拟环境，所以我将chromedriver.exe放于原环境和虚拟环境中都进行了尝试，发现都可以，只要是于python.exe在同一目录下即可。
